# Math Quiz Application

This is a command-line math quiz application that adapts to the user's skill level.

## Features

*   Generates math problems (addition, subtraction, multiplication, division).
*   Tracks user's progress.
*   Adapts to the user's skill level using a logic base and logistic regression model hybrid.

## Files

*   `main.py`: The main application file.
*   `tracker.py`: Handles user progress tracking.
*   `adaptive_engine.py`: The adaptive engine for question difficulty.
*   `puzzle_generator.py`: Generates the math puzzles.
*   `requirements.txt`: Lists the Python dependencies.

## Usage Instructions

*   Install all the requirements in the requirements.txt file
*   Run the main.py for command line interface
